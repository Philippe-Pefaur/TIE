# Sistema de seguimiento de consumo, generación y pérdida eléctrica

&emsp;Este repositorio contiene un sistema mediante el cual se puede, de manera local, presentar datos de flujo de energía en un sistema de generación eléctrica en el hogar recogidos por un sistema de sensores con Arduino. Los datos son presentados mediante una app web para facilitar la presentación.

# Ejecución

&emsp;Para iniciar el proyecto abra línea de comandos en el directorio del repositorio y ejecute:

```bash
npm run install
npm run start
```
